## 🧪 测试功能

### 1. 测试配置加载
确保项目根目录有 `config.yaml`：

```yaml
version: "1.0"
accounts:
  - name: test-account
    monitor_chats:
      - "Saved Messages"
    rules:
      - keywords: ["hello"]
        reply_text: "Hi there!"
```

### 2. 测试启动/停止
1. 点击账号的"▶️ 启动"按钮
2. 查看日志输出
3. 点击"⏹ 停止"按钮

### 3. 测试实时日志
- 日志会自动出现在底部日志查看器
- 最多保留最近 100 条日志

## ⚠️ 注意事项

1. **首次启动**可能需要几秒钟初始化
2. **确保 Python Teleflow CLI 可用**: `python -m teleflow --version`
3. **配置文件路径**: 默认为 `config.yaml`（可在代码中修改）
4. **日志限制**: 最多保留最近 100 条（防止内存溢出）
